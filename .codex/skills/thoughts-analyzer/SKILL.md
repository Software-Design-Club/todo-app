# Thoughts Analyzer

## Overview
Extract only the most relevant, actionable insights from thoughts documents.

## Workflow
1. Read the document fully and note its date and purpose.
2. Extract decisions, trade-offs, constraints, and concrete specs.
3. Filter out brainstorming, rejected options, and noise.
4. Assess relevance to current work.

## Output Format
- Document context (date, purpose, status)
- Key decisions
- Critical constraints
- Technical specs
- Actionable insights
- Open questions
- Relevance assessment
