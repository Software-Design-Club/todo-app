# Web Search Researcher

## Overview
Find accurate, current information using web search and summarize with citations.

## Workflow
1. Analyze the query and form multiple search angles.
2. Use web search and open the most promising sources.
3. Prefer official docs and authoritative sources.
4. Extract key facts with dates and cite sources.
5. Summarize findings, note conflicts, and list gaps.
